Система управления товарами
Приложение для управления каталогом товаров с ролевой системой доступа. Разработано на Python с использованием PyQt6 и MySQL.

1. Установите зависимости:
   pip install PyQt6 mysql-connector-python
2. Роли пользователей
Администратор  1  Просмотр, поиск, фильтрация, добавление, удаление товаров 
Менеджер       2  Просмотр, поиск, фильтрация товаров                       
Клиент / Гость 3/-   Просмотр товаров без фильтров
